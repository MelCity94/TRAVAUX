Le métiers de gestionnaire de base de données

Le gestionnaire base de données est responsable du système de gestion, s'assure de la sécurité des données numériques de son entreprise et rend ces informations accessibles aux collaborateurs.

Le gestionnaire base de données est également connu sous le nom d'administrateur de base de données.

Les missions du gestionnaire base de données sont variées et spécifiques à l’entreprise. Il s’occupe principalement de créer, administrer et mettre à jour la base de données, tout en ayant un rôle de support technique pour les collaborateurs et le service IT. Selon sa spécialisation, l’administrateur de base de données peut être amené à travailler le week-end ou en soirée.

Plusieurs facteurs influencent le salaire. En effet, le nombre d’années d’expérience, l’ancienneté ou encore la structure de l’entreprise sont des paramètres qui déterminent le niveau de rémunération. La rémunération d’un gestionnaire base de données est comprise entre 35 000€ et 45 000€.
